üéØ What are Conditional Statements?

Conditional statements allow your program to make decisions based on certain conditions. Think of them as the brain of your program!

Imagine you're deciding what to wear:

üí° Key Concept: Indentation

Python uses indentation (spaces at the beginning of a line) to define code blocks. Always use 4 spaces inside your conditionals!

üî∏ The if-elif-else Chain

When you have multiple conditions to check, use elif (short for "else if").

üìù Syntax

Python 
if condition1:
    # code if condition1 is True
elif condition2:
    # code if condition2 is True
elif condition3:
    # code if condition3 is True
else:
    # code if ALL conditions are False

‚ö†Ô∏è Important!

Python checks conditions from top to bottom. Once a condition is True, it runs that code and skips the rest!

    # runs if outer condition is False

‚≠ê The match-case Statement

New in Python 3.10!

The match-case statement provides a cleaner way to handle multiple specific values. It's similar to switch in other languages.

üìù Syntax

Python 
match variable:
    case value1:
        # code for value1
    case value2:
        # code for value2
    case value3 | value4:
        # code for value3 OR value4
    case _:
        # default case (like 'else')

üéØ Example: Day of the Week

Python 
day = "Monday"

match day:
    case "Monday":
        print("üò´ Start of the work week...")
    case "Tuesday" | "Wednesday" | "Thursday":
        print("üíº Keep working hard!")
    case "Friday":
        print("üéâ TGIF! Almost weekend!")
    case "Saturday" | "Sunday":
        print("üéÆ Weekend! Time to relax!")
    case _:
        print("‚ùì That's not a valid day!")

üéØ Example: HTTP Status Codes

Python 
status_code = 404

match status_code:
    case 200:
        print("‚úÖ OK - Request successful!")
    case 301 | 302:
        print("üîÑ Redirect - Page moved!")
    case 404:
        print("‚ùå Not Found - Page doesn't exist!")
    case 500:
        print("üí• Server Error!")
    case _:
        print(f"Unknown status: {status_code}")

üîÑ match vs if-elif: When to Use?

Use match when:

  • Checking ONE variable against many specific values
  • Values are exact matches (not ranges)
  • You want cleaner, more readable code

Use if-elif when:

  • Checking ranges (like score >= 90)
  • Complex conditions with and/or
  • Using Python versions before 3.10
