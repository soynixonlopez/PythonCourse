üéØ File Handling in Python

File handling lets your programs save data permanently and read existing files!

üìã File Modes

ModeDescription
"r"Read (default) - File must exist
"w"Write - Creates/overwrites file
"a"Append - Creates file if not exists
"r+"Read and Write
"x"Create - Fails if file exists

üìù Basic open() Syntax

Python 
file = open("filename.txt", "r")  # Open file
# ... do something with file ...
file.close()  # Always close!

‚ö†Ô∏è Always Close Files!

Open files use system resources. Always close them, or use with (recommended).

üìñ Reading Files

üìù Read Entire File

Python 
# Read all content at once
file = open("data.txt", "r")
content = file.read()
print(content)
file.close()

üìù Read Line by Line

Python 
# Read one line
line = file.readline()

# Read all lines as list
lines = file.readlines()

# Loop through lines
for line in file:
    print(line.strip())  # strip() removes \n

üìù Read Specific Amount

Python 
# Read first 10 characters
chunk = file.read(10)

‚úèÔ∏è Writing Files

üìù Write Mode (Overwrites!)

Python 
file = open("output.txt", "w")
file.write("Hello, World!\n")
file.write("Second line\n")
file.close()

üìù Append Mode

Python 
# Add to existing file
file = open("log.txt", "a")
file.write("New log entry\n")
file.close()

üìù Write Multiple Lines

Python 
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
file.writelines(lines)

‚ö†Ô∏è "w" Mode Erases Content!

Opening with "w" deletes all existing content. Use "a" to append.

üîí The with Statement

The with statement automatically closes the file, even if errors occur!

üìù Recommended Way

Python 
# Reading
with open("data.txt", "r") as file:
    content = file.read()
    print(content)
# File automatically closed here!

# Writing
with open("output.txt", "w") as file:
    file.write("Hello!")
# File automatically closed!

üìù Multiple Files

Python 
with open("input.txt") as infile, open("output.txt", "w") as outfile:
    content = infile.read()
    outfile.write(content.upper())

üí° Always Use with!

It's safer, cleaner, and the Pythonic way to handle files.
