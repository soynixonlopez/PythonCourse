ğŸ¯ What are Tuples?

A tuple is like a list, but immutable (cannot be changed after creation). Use tuples when you need data that shouldn't be modified!

ğŸŒ Real-Life Analogy

Think of a tuple like a sealed envelope:

  • Once sealed, contents cannot be changed
  • You can read what's inside
  • If you need changes, create a new envelope

ğŸ’¡ When to Use Tuples?

  • Data that shouldn't change (coordinates, RGB colors)
  • Dictionary keys (lists can't be keys)
  • Returning multiple values from functions
  • When you need slightly better performance

ğŸ“¦ Tuple Packing & Unpacking

ğŸ“ Packing (creating a tuple)

Python 
# Packing - multiple values into one tuple
person = "Alice", 25, "Developer"
print(person)  # ("Alice", 25, "Developer")

ğŸ“ Unpacking (extracting values)

Python 
person = ("Alice", 25, "Developer")

# Unpacking into variables
name, age, job = person

print(name)  # "Alice"
print(age)   # 25
print(job)   # "Developer"

ğŸ“ Unpacking with * (asterisk)

Python 
numbers = (1, 2, 3, 4, 5)

# First, *middle, last
first, *middle, last = numbers

print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# Ignore some values with _
a, _, _, d, e = numbers
print(a, d, e)  # 1 4 5

ğŸ› ï¸ Tuple Methods

Since tuples are immutable, they only have 2 methods!

ğŸ“‹ Available Methods

Method Description Example
count(x) Count occurrences of x t.count(2)
index(x) Find index of first x t.index("a")

ğŸ“ Examples

Python 
numbers = (1, 2, 2, 3, 2, 4)

# Count occurrences
print(numbers.count(2))  # 3

# Find index
print(numbers.index(3))  # 3

# Other operations (not methods)
print(len(numbers))   # 6
print(min(numbers))   # 1
print(max(numbers))   # 4
print(sum(numbers))   # 14
print(2 in numbers)  # True

ğŸ“ Converting Tuples

Python 
# Tuple to List (to modify)
my_tuple = (1, 2, 3)
my_list = list(my_tuple)
my_list.append(4)
new_tuple = tuple(my_list)
print(new_tuple)  # (1, 2, 3, 4)

# Concatenate tuples
t1 = (1, 2)
t2 = (3, 4)
t3 = t1 + t2  # (1, 2, 3, 4)

# Repeat tuple
t4 = ("Hi",) * 3  # ("Hi", "Hi", "Hi")
