üéØ What are Strings?

Strings are sequences of characters. They can contain letters, numbers, symbols, and even emojis! In Python, strings are immutable (cannot be changed after creation).

üåç Real-Life Analogy

Think of a string like a necklace of beads:

  • Each bead is a character
  • Beads have positions (index)
  • You can look at any bead by its position
  • You can take a section of beads (slicing)

üìã Creating Strings

Python 
# Single quotes
name = 'Alice'

# Double quotes
greeting = "Hello, World!"

# Triple quotes (multi-line)
poem = """Roses are red,
Violets are blue,
Python is awesome,
And so are you!"""

# String with quotes inside
quote = "She said 'Hello!'"
quote2 = 'He said "Hi!"'

üìã String Properties

Property Description Example
Immutable Cannot be changed after creation s[0] = 'X' ‚ùå Error
Indexed Each character has a position "Hello"[0] ‚Üí 'H'
Iterable Can loop through characters for char in "Hi":
Concatenatable Can join strings together "Py" + "thon" ‚Üí 'Python'

üí° Escape Characters

Special characters start with backslash \:

  • \n - New line
  • \t - Tab
  • \\ - Backslash
  • \' - Single quote
  • \" - Double quote

üî¢ String Indexing

Each character in a string has a position number called an index. Python uses zero-based indexing (starts from 0).

üìä Visual Index Example: "PYTHON"

P Y T H O N
0 1 2 3 4 5
-6 -5 -4 -3 -2 -1

üìù Positive Indexing (from start)

Python 
word = "PYTHON"

print(word[0])  # P (first character)
print(word[1])  # Y (second character)
print(word[5])  # N (last character)

üìù Negative Indexing (from end)

Python 
word = "PYTHON"

print(word[-1])  # N (last character)
print(word[-2])  # O (second to last)
print(word[-6])  # P (first character)

‚ö†Ô∏è IndexError

Accessing an index that doesn't exist causes an error:

word = "PYTHON"  # length is 6
print(word[10])  # ‚ùå IndexError!

‚úÇÔ∏è String Slicing

Slicing lets you extract a portion of a string. Think of it as cutting a piece from the middle!

üìù Syntax: string[start:stop:step]

Parameter Description Default
start Where to start (inclusive) 0 (beginning)
stop Where to stop (exclusive) end of string
step How many to skip 1

üéØ Basic Slicing Examples

Python 
text = "Hello, World!"

# Get characters 0 to 4 (5 is excluded)
print(text[0:5])   # "Hello"

# From index 7 to end
print(text[7:])    # "World!"

# From start to index 5
print(text[:5])    # "Hello"

# Get the whole string
print(text[:])     # "Hello, World!"

üéØ Slicing with Step

Python 
text = "ABCDEFGHIJ"

# Every second character
print(text[::2])    # "ACEGI"

# Every third character
print(text[::3])    # "ADGJ"

# Reverse the string!
print(text[::-1])   # "JIHGFEDCBA"

üéØ Negative Slicing

Python 
text = "Hello, World!"

# Last 6 characters
print(text[-6:])     # "World!"

# Everything except last character
print(text[:-1])     # "Hello, World"

# From -6 to -1 (exclusive)
print(text[-6:-1])   # "World"

üí° Common Slice Patterns

  • s[::-1] - Reverse string
  • s[1:] - Remove first character
  • s[:-1] - Remove last character
  • s[::2] - Every other character

üõ†Ô∏è String Methods

Python strings come with many built-in methods (functions attached to strings) that make text manipulation easy!

üìã Case Methods

Python 
text = "hello WORLD"

print(text.upper())       # "HELLO WORLD"
print(text.lower())       # "hello world"
print(text.capitalize())  # "Hello world"
print(text.title())       # "Hello World"
print(text.swapcase())    # "HELLO world"

üìã Search Methods

Python 
text = "Hello, World!"

# Find position of substring
print(text.find("World"))     # 7
print(text.find("Python"))    # -1 (not found)

# Check if contains
print("World" in text)       # True
print(text.startswith("Hello")) # True
print(text.endswith("!"))      # True

# Count occurrences
print(text.count("o"))        # 2

üìã Modify Methods

Python 
# Replace
text = "Hello, World!"
print(text.replace("World", "Python"))  # "Hello, Python!"

# Strip whitespace
messy = "   hello   "
print(messy.strip())   # "hello"
print(messy.lstrip())  # "hello   " (left only)
print(messy.rstrip())  # "   hello" (right only)

# Split into list
sentence = "apple,banana,cherry"
print(sentence.split(","))  # ['apple', 'banana', 'cherry']

# Join list into string
fruits = ['apple', 'banana', 'cherry']
print(", ".join(fruits))  # "apple, banana, cherry"

üìã Check Methods (return True/False)

Python 
print("hello".isalpha())    # True (all letters)
print("12345".isdigit())    # True (all digits)
print("hello123".isalnum()) # True (letters or digits)
print("   ".isspace())       # True (all whitespace)
print("HELLO".isupper())    # True (all uppercase)
print("hello".islower())    # True (all lowercase)

üí° Remember!

String methods return new strings. They don't modify the original!

name = "alice"
name.upper()  # Returns "ALICE" but name is still "alice"
name = name.upper()  # Now name is "ALICE"

üé® f-Strings (Formatted String Literals)

f-Strings are the modern way to format strings in Python (3.6+). They're fast, readable, and powerful!

üìù Basic f-String Syntax

Python 
name = "Alice"
age = 25

# f-string (modern way)
print(f"Hello, {name}! You are {age} years old.")
# Output: Hello, Alice! You are 25 years old.

üéØ Expressions Inside f-Strings

Python 
price = 19.99
quantity = 3

# Math inside f-string
print(f"Total: ${price * quantity}")
# Output: Total: $59.97

# Method calls
name = "bob"
print(f"Hello, {name.upper()}!")
# Output: Hello, BOB!

# Conditional expression
score = 85
print(f"Result: {'Pass' if score >= 60 else 'Fail'}")
# Output: Result: Pass

üéØ Number Formatting

Python 
pi = 3.14159265359
big_num = 1234567890
percent = 0.875

# Decimal places
print(f"Pi: {pi:.2f}")           # Pi: 3.14
print(f"Pi: {pi:.4f}")           # Pi: 3.1416

# Thousands separator
print(f"Number: {big_num:,}")    # Number: 1,234,567,890

# Percentage
print(f"Score: {percent:.1%}")   # Score: 87.5%

# Padding
num = 42
print(f"ID: {num:05d}")          # ID: 00042 (pad with zeros)
print(f"ID: {num:>10}")         # ID:         42 (right align)

üìã Old vs New Formatting

Python 
name = "Alice"
age = 25

# Old way 1: % formatting
print("Hello, %s! Age: %d" % (name, age))

# Old way 2: .format()
print("Hello, {}! Age: {}".format(name, age))

# Modern way: f-strings ‚úÖ
print(f"Hello, {name}! Age: {age}")

üí° Why f-Strings?

  • More readable - variables are right in the string
  • Faster than other methods
  • Can include any Python expression
  • Less error-prone than % formatting
