Modules & Imports

Organize and reuse code with modules! üì¶

üéØ What are Modules?

A module is a Python file (.py) containing code you can reuse. Python comes with many built-in modules!

üåç Real-Life Analogy

Think of modules like a toolbox:

  • Each tool (function) serves a purpose
  • You pick the tools you need
  • No need to build every tool yourself

üìã Why Use Modules?

BenefitDescription
OrganizationSplit code into logical files
ReusabilityUse code in multiple programs
NamespaceAvoid naming conflicts
CommunityUse libraries made by others

üì• Import Methods

üìù Basic Import

Python 
import math

print(math.pi)         # 3.14159...
print(math.sqrt(16))   # 4.0
print(math.ceil(4.2))  # 5

üìù Import Specific Items

Python 
from math import pi, sqrt

print(pi)       # 3.14159... (no math. prefix!)
print(sqrt(16))  # 4.0

üìù Import with Alias

Python 
import math as m

print(m.pi)      # Use short name
print(m.sqrt(16)) # 4.0

# Common aliases
import numpy as np
import pandas as pd

üìù Import All (Not Recommended)

Python 
from math import *  # Imports everything - avoid!

print(pi)  # Works, but where did pi come from?

‚ö†Ô∏è Avoid from x import *

It pollutes your namespace and makes it unclear where things come from.

üêç Useful Standard Library Modules

üìù math - Mathematical Functions

Python 
import math

math.pi          # 3.14159...
math.sqrt(16)    # 4.0
math.ceil(4.2)   # 5 (round up)
math.floor(4.8)  # 4 (round down)
math.pow(2, 3)   # 8.0 (2^3)

üìù random - Random Numbers

Python 
import random

random.random()           # 0.0 to 1.0
random.randint(1, 10)     # 1 to 10 (inclusive)
random.choice(["a", "b"]) # Random element
random.shuffle(my_list)   # Shuffle in place

üìù datetime - Date and Time

Python 
from datetime import datetime, date

now = datetime.now()
print(now)              # Current date and time
print(now.year)          # Current year
print(date.today())      # Today's date

üìù os - Operating System

Python 
import os

os.getcwd()            # Current directory
os.listdir(".")        # List files in directory
os.path.exists("file.txt")  # Check if exists
