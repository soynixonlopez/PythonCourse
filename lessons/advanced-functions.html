üì• *args and **kwargs

These allow functions to accept a variable number of arguments!

üìù *args (Positional Arguments)

Python 
def add_all(*args):
    """Add any number of values"""
    total = 0
    for num in args:
        total += num
    return total

print(add_all(1, 2))           # 3
print(add_all(1, 2, 3, 4, 5))  # 15
print(add_all(10))              # 10

üìù **kwargs (Keyword Arguments)

Python 
def introduce(**kwargs):
    """Print all key-value pairs"""
    for key, value in kwargs.items():
        print(f"{key}: {value}")

introduce(name="Alice", age=25, city="NYC")
# name: Alice
# age: 25
# city: NYC

üìù Combining All Types

Python 
def complex_func(a, b, *args, **kwargs):
    print(f"a: {a}, b: {b}")
    print(f"args: {args}")
    print(f"kwargs: {kwargs}")

complex_func(1, 2, 3, 4, x=10, y=20)
# a: 1, b: 2
# args: (3, 4)
# kwargs: {'x': 10, 'y': 20}

üí° Order Matters!

Parameters must be in this order: regular ‚Üí *args ‚Üí **kwargs

‚ö° Lambda Functions

Lambda functions are small, anonymous (nameless) functions defined in one line.

üìù Syntax

Python 
lambda arguments: expression

# Regular function
def add(a, b):
    return a + b

# Same as lambda
add = lambda a, b: a + b
print(add(3, 5))  # 8

üìù Common Use Cases

Python 
# With sorted()
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
by_score = sorted(students, key=lambda x: x[1])
# [('Charlie', 78), ('Alice', 85), ('Bob', 92)]

# With map()
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
# [1, 4, 9, 16]

# With filter()
evens = list(filter(lambda x: x % 2 == 0, nums))
# [2, 4]

üí° When to Use Lambda?

  • Simple, one-time operations
  • As arguments to functions like sort(), map(), filter()
  • When a full function is overkill

üéØ Example: Factorial

Python 
def factorial(n):
    # Base case
    if n == 0 or n == 1:
        return 1
    # Recursive case
    return n * factorial(n - 1)

print(factorial(5))  # 120 (5*4*3*2*1)

üéØ Example: Countdown

Python 
def countdown(n):
    if n <= 0:
        print("Blast off! üöÄ")
    else:
        print(n)
        countdown(n - 1)

countdown(5)  # 5, 4, 3, 2, 1, Blast off!

‚ö†Ô∏è Always Need a Base Case!

Without a base case, recursion continues forever ‚Üí "RecursionError"
