Save & Load Tasks (File Persistence)

Now let's make tasks persist between program runs using JSON files!

ğŸ“‹ Task A: save_tasks() Function

Create a function that:

  • Import json at the top of your file
  • Opens a file called "tasks.json" in write mode ("w")
  • Uses json.dump(tasks, f) to save the list
  • Uses try/except to handle errors
import json

FILENAME = "tasks.json"

def save_tasks():
    try:
        with open(FILENAME, "w") as f:
            json.dump(tasks, f, indent=2)
    except IOError as e:
        print(f"Error saving: {e}")

ğŸ“‹ Task B: load_tasks() Function

Create a function that:

  • Uses global tasks to modify the global list
  • Opens "tasks.json" in read mode ("r")
  • Uses json.load(f) to load tasks
  • Handles FileNotFoundError (file doesn't exist yet)
  • Handles json.JSONDecodeError (corrupt file)
def load_tasks():
    global tasks
    
    try:
        with open(FILENAME, "r") as f:
            tasks = json.load(f)
            print(f"Loaded {len(tasks)} task(s)")
    except FileNotFoundError:
        tasks = []
        print("Starting fresh - no saved tasks")
    except json.JSONDecodeError:
        tasks = []
        print("Error reading file - starting fresh")
