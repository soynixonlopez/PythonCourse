üî§ What are Data Types?

In Python, every value has a type. The type tells Python what kind of data it is and what operations can be done with it.

üåç Real-Life Analogy

Think of data types like different containers:

  • üìù Strings = A notebook (holds text)
  • üî¢ Integers = A counter (holds whole numbers)
  • üíß Floats = A measuring cup (holds decimal numbers)
  • ‚úÖ Booleans = A light switch (on/off, true/false)

üìã Python's Basic Data Types

Type Description Example
str Text (strings) "Hello", 'Python'
int Whole numbers 42, -10, 0
float Decimal numbers 3.14, -0.5, 2.0
bool True or False True, False

üîç Check the Type with type()

Python 
# Use type() to check data types
print(type("Hello"))     # <class 'str'>
print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type(True))        # <class 'bool'>

üìù Strings (str)

A string is a sequence of characters (text). Always wrap strings in quotes!

‚ú® Creating Strings

Python 
# Single quotes
name = 'Alex'

# Double quotes (same thing!)
greeting = "Hello, World!"

# Triple quotes (for multiple lines)
poem = """Roses are red,
Violets are blue,
Python is awesome,
And so are you!"""

# Empty string
empty = ""

üîó String Concatenation (+)

Combine strings using the + operator:

Python 
first_name = "John"
last_name = "Doe"

# Combine strings
full_name = first_name + " " + last_name
print(full_name)  # John Doe

# Repeat strings with *
laugh = "ha" * 3
print(laugh)  # hahaha

üîç Comparison Operators

Comparisons return boolean values:

Python 
x = 10
y = 5

print(x > y)    # True  (10 is greater than 5)
print(x < y)    # False (10 is not less than 5)
print(x == y)   # False (10 does not equal 5)
print(x != y)   # True  (10 is not equal to 5)
print(x >= 10)  # True  (10 is >= 10)
print(x <= 5)   # False (10 is not <= 5)

üí° Boolean Values

In Python, some values are considered "truthy" or "falsy":

  • Falsy: False, 0, "" (empty string), None
  • Truthy: Everything else!

üîÑ Type Conversion

Sometimes you need to convert data from one type to another. Python provides built-in functions for this!

üîß Conversion Functions

FunctionConverts toExample
str()Stringstr(42) ‚Üí "42"
int()Integerint("42") ‚Üí 42
float()Floatfloat("3.14") ‚Üí 3.14
bool()Booleanbool(1) ‚Üí True

üìù Examples

Python 
# String to Number
age_str = "25"
age_int = int(age_str)
print(age_int + 5)  # 30

# Number to String
score = 100
message = "Your score is: " + str(score)
print(message)  # Your score is: 100

# Float to Integer (removes decimals!)
price = 19.99
whole_price = int(price)
print(whole_price)  # 19

# Integer to Float
number = 42
decimal = float(number)
print(decimal)  # 42.0
