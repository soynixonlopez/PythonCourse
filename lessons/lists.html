ğŸ¯ What are Lists?

A list is an ordered collection of items. Lists can hold any type of data and you can add, remove, or change items!

ğŸŒ Real-Life Analogy

Think of a list like a shopping list:

  • Items are in a specific order
  • You can add new items
  • You can remove items
  • You can check what's on the list
  • Items can be repeated

ğŸ“ Creating Lists

Python

# List of numbers

# List of strings

# Mixed types
mixed = [1, "hello", 3.14, True]

# Nested lists (list of lists)
matrix = [[1, 2], [3, 4], [5, 6]]

# Using list() constructor
letters = list("hello")  # ['h', 'e', 'l', 'l', 'o']

ğŸ“ Indexing

Python 
fruits = ["apple", "banana", "cherry", "date"]

# Positive indexing
print(fruits[0])   # "apple" (first)
print(fruits[2])   # "cherry" (third)

# Negative indexing
print(fruits[-1])  # "date" (last)
print(fruits[-2])  # "cherry" (second to last)

ğŸ“ Slicing

Python 
numbers = [0, 1, 2, 3, 4, 5]

print(numbers[1:4])   # [1, 2, 3]
print(numbers[:3])    # [0, 1, 2]
print(numbers[3:])    # [3, 4, 5]
print(numbers[::2])   # [0, 2, 4] (every other)
print(numbers[::-1])  # [5, 4, 3, 2, 1, 0] (reversed)

ğŸ“ Nested List Access

Python 
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(matrix[0])       # [1, 2, 3] (first row)
print(matrix[0][0])    # 1 (first row, first item)
print(matrix[1][2])    # 6 (second row, third item)
print(matrix[-1][-1])  # 9 (last row, last item)

âœï¸ Modifying Lists

Lists are mutable - you can change, add, and remove items!

ğŸ“ Changing Items

Python 
fruits = ["apple", "banana", "cherry"]

# Change single item
fruits[1] = "blueberry"
print(fruits)  # ["apple", "blueberry", "cherry"]

# Change multiple items with slicing
fruits[0:2] = ["apricot", "blackberry"]
print(fruits)  # ["apricot", "blackberry", "cherry"]

ğŸ“ Adding Items

Python 
fruits = ["apple", "banana"]

# append() - add to end
fruits.append("cherry")
print(fruits)  # ["apple", "banana", "cherry"]

# insert() - add at specific position
fruits.insert(1, "blueberry")
print(fruits)  # ["apple", "blueberry", "banana", "cherry"]

# extend() - add multiple items
fruits.extend(["date", "elderberry"])
print(fruits)  # ["apple", "blueberry", "banana", "cherry", "date", "elderberry"]

# Concatenation with +
more_fruits = fruits + ["fig", "grape"]

ğŸ“ Removing Items

Python 
fruits = ["apple", "banana", "cherry", "banana"]

# remove() - remove first occurrence of value
fruits.remove("banana")
print(fruits)  # ["apple", "cherry", "banana"]

# pop() - remove by index (default: last item)
last = fruits.pop()
print(last)    # "banana"
print(fruits)  # ["apple", "cherry"]

first = fruits.pop(0)
print(first)   # "apple"

# del - delete by index or slice
nums = [1, 2, 3, 4, 5]
del nums[0]      # [2, 3, 4, 5]
del nums[1:3]   # [2, 5]

# clear() - remove all items
nums.clear()
print(nums)  # []

âš ï¸ Common Errors

  • remove() raises ValueError if item not found
  • pop() raises IndexError if list is empty or index out of range

ğŸ¯ Sorting Examples

Python 
numbers = [3, 1, 4, 1, 5, 9]

# Sort ascending (modifies original)
numbers.sort()
print(numbers)  # [1, 1, 3, 4, 5, 9]

# Sort descending
numbers.sort(reverse=True)
print(numbers)  # [9, 5, 4, 3, 1, 1]

# sorted() returns NEW list (original unchanged)
original = [3, 1, 2]
new_sorted = sorted(original)
print(original)    # [3, 1, 2] (unchanged)
print(new_sorted)  # [1, 2, 3]

# Sort strings alphabetically
words = ["banana", "Apple", "cherry"]
words.sort()  # ['Apple', 'banana', 'cherry'] (case-sensitive!)
words.sort(key=str.lower)  # Case-insensitive sort

ğŸ¯ Finding Items

Python 
fruits = ["apple", "banana", "cherry", "banana"]

# Find index of item
print(fruits.index("banana"))  # 1 (first occurrence)

# Count occurrences
print(fruits.count("banana"))  # 2

# Check membership
print("apple" in fruits)      # True

# Min, max, sum (for numbers)
nums = [1, 5, 3, 9, 2]
print(min(nums))  # 1
print(max(nums))  # 9
print(sum(nums))  # 20

ğŸ’¡ Copy vs Reference

Be careful when copying lists!

a = [1, 2, 3]
b = a           # b points to SAME list!
b[0] = 99
print(a)  # [99, 2, 3] - a changed too!

c = a.copy()     # c is a NEW copy
c[0] = 1
print(a)  # [99, 2, 3] - a unchanged

ğŸ”„ Looping Through Lists

ğŸ“ Basic For Loop

Python 
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(f"I like {fruit}")

# Output:
# I like apple
# I like banana
# I like cherry

ğŸ“ Loop with Index (enumerate)

Python 
fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

# Output:
# 0: apple
# 1: banana
# 2: cherry

ğŸ“ Loop with range()

Python 
fruits = ["apple", "banana", "cherry"]

for i in range(len(fruits)):
    print(f"{i}: {fruits[i]}")

ğŸ“ While Loop

Python 
fruits = ["apple", "banana", "cherry"]
i = 0

while i < len(fruits):
    print(fruits[i])
    i += 1

ğŸ“ List Comprehension (Advanced)

Python 
# Create new list from existing
numbers = [1, 2, 3, 4, 5]

# Square each number
squares = [x**2 for x in numbers]
print(squares)  # [1, 4, 9, 16, 25]

# Filter even numbers
evens = [x for x in numbers if x % 2 == 0]
print(evens)   # [2, 4]

# Transform strings
fruits = ["apple", "banana"]
upper_fruits = [f.upper() for f in fruits]
print(upper_fruits)  # ["APPLE", "BANANA"]

ğŸ’¡ Which Loop to Use?

  • for item in list - When you just need the items
  • enumerate() - When you need index AND item
  • range(len()) - When you need to modify items
  • List comprehension - When creating a new list
