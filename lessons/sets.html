üéØ What are Sets?

A set is an unordered collection of unique items. No duplicates allowed!

üåç Real-Life Analogy

Think of a set like a bag of unique marbles:

  • No two marbles are exactly the same
  • No particular order
  • Easy to check "do I have a red marble?"

üìù Creating Sets

Python 
# Using curly braces
fruits = {"apple", "banana", "cherry"}

# Duplicates are automatically removed!
numbers = {1, 2, 2, 3, 3, 3}
print(numbers)  # {1, 2, 3}

# Empty set (NOT {} - that's a dict!)
empty = set()

# From list
unique = set([1, 1, 2, 2, 3])  # {1, 2, 3}

üìã Set Properties

PropertyDescription
UnorderedNo index, no guaranteed order
UniqueNo duplicate values
MutableCan add/remove items
No IndexingCannot access by position
Fast Lookupin operator is O(1)

üí° When to Use Sets?

  • Remove duplicates from a list
  • Fast membership testing
  • Mathematical set operations (union, intersection)
  • Tracking unique items

üõ†Ô∏è Set Methods

üìù Adding & Removing Items

Python 
fruits = {"apple", "banana"}

# add() - add one item
fruits.add("cherry")

# update() - add multiple items
fruits.update(["date", "elderberry"])

# remove() - remove (error if not found)
fruits.remove("banana")

# discard() - remove (no error if not found)
fruits.discard("not_here")  # No error!

# pop() - remove random item
item = fruits.pop()

# clear() - remove all
fruits.clear()

üìã All Set Methods

MethodDescription
add(x)Add item x
remove(x)Remove x (error if missing)
discard(x)Remove x (no error if missing)
pop()Remove and return arbitrary item
clear()Remove all items
copy()Return shallow copy

üîó Set Operations

üìù Mathematical Set Operations

Python 
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Union - all items from both sets
print(a | b)  # {1, 2, 3, 4, 5, 6}
print(a.union(b))

# Intersection - items in BOTH sets
print(a & b)  # {3, 4}
print(a.intersection(b))

# Difference - items in a but NOT in b
print(a - b)  # {1, 2}
print(a.difference(b))

# Symmetric Difference - items in either, but NOT both
print(a ^ b)  # {1, 2, 5, 6}
print(a.symmetric_difference(b))

üìù Subset & Superset

Python 
a = {1, 2}
b = {1, 2, 3, 4}

# Is a a subset of b? (all a items in b)
print(a.issubset(b))    # True
print(a <= b)            # True

# Is b a superset of a?
print(b.issuperset(a))  # True
print(b >= a)            # True

# Are they disjoint? (no common elements)
c = {5, 6}
print(a.isdisjoint(c))  # True

üéØ Practical Example: Remove Duplicates

Python 
numbers = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(numbers))
print(unique)  # [1, 2, 3, 4]
