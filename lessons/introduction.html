üñ®Ô∏è The print() Function

The print() function is your window to communicate with the world! It displays information on the screen.

üìù Basic Syntax

Python 
print("Your message here")

üéØ Printing Multiple Things

You can print several items separated by commas:

Python 
# Printing multiple items
print("Hello", "World", "!")

# Output: Hello World !

# Printing text and numbers
print("I am", 15, "years old")

# Output: I am 15 years old

üî¢ Printing Numbers

Numbers don't need quotes:

Python 
# Printing numbers (no quotes needed!)
print(42)
print(3.14159)

# You can even do math!
print(10 + 5)      # Output: 15
print(100 - 25)    # Output: 75
print(7 * 8)       # Output: 56
print(20 / 4)      # Output: 5.0

üÜï Empty Lines

Use print() with nothing inside to create a blank line:

Python

print("Line 3")

üí° Pro Tip: Using Quotes

Both single ' and double " quotes work the same:

print("Hello!")   # Double quotes
print('Hello!')   # Single quotes - same result!

Use double quotes when your text contains an apostrophe:

print("I'm learning Python!")  # ‚úì Works!
print('I'm learning Python!')  # ‚úó Error!

üí¨ Comments in Python

Comments are notes for humans that Python ignores. They help explain your code!

üìù Single-Line Comments

Use the # symbol to create a comment:

Python 
# This is a comment - Python ignores this line
print("Hello!")  # You can also comment at the end of a line

# Comments help explain what your code does
# They're like sticky notes for programmers!

üìÑ Multi-Line Comments

For longer explanations, use triple quotes:

Python 
"""
This is a multi-line comment.
You can write as many lines as you want.
Python will ignore all of this!
"""

print("This code runs normally")

üéØ Why Use Comments?

  • Explain complex code - Help others (and future you!) understand
  • Leave reminders - "TODO: fix this later"
  • Temporarily disable code - Without deleting it
  • Document your program - What does it do? How does it work?

üéØ Good vs Bad Comments

‚úÖ Good Comments

# Calculate the total price with 10% tax
total = price * 1.10

# Check if user is old enough to vote
if age >= 18:

‚ùå Bad Comments

# Print hello
print("Hello")

# Add 1 to x
x = x + 1

üí° Pro Tip

Write comments that explain WHY you're doing something, not WHAT the code does. The code itself shows what it does!
