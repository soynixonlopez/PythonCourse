üéØ What are Comprehensions?

Comprehensions are a concise, Pythonic way to create collections (lists, dicts, sets) from existing iterables in a single line!

üåç Before vs After

Traditional Way
squares = []
for x in range(5):
    squares.append(x ** 2)
Comprehension Way
squares = [x ** 2 for x in range(5)]

Both produce: [0, 1, 4, 9, 16]

BenefitDescription
ConciseOne line instead of multiple

üìã List Comprehension

üìù Basic Syntax

Python 
[expression for item in iterable]

üéØ Examples

Python 
# Square numbers
squares = [x**2 for x in range(5)]
# [0, 1, 4, 9, 16]

# Double each number
nums = [1, 2, 3]
doubled = [n * 2 for n in nums]
# [2, 4, 6]

# Convert to uppercase
words = ["hello", "world"]
upper = [w.upper() for w in words]
# ["HELLO", "WORLD"]

üìù With Condition (Filtering)

Python 
[expression for item in iterable if condition]

# Only even numbers
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]

# Words longer than 3 characters
words = ["hi", "hello", "hey", "howdy"]
long_words = [w for w in words if len(w) > 3]
# ["hello", "howdy"]

üìù With if-else

Python 
# Note: if-else goes BEFORE for
[expression_if if condition else expression_else for item in iterable]

# Mark even/odd
labels = ["even" if x % 2 == 0 else "odd" for x in range(5)]
# ["even", "odd", "even", "odd", "even"]

üìñ Dictionary Comprehension

üìù Basic Syntax

Python 
{key_expr: value_expr for item in iterable}

üéØ Examples

Python 
# Number to its square
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Word to length
words = ["hi", "hello", "hey"]
lengths = {w: len(w) for w in words}
# {"hi": 2, "hello": 5, "hey": 3}

# From two lists
keys = ["a", "b", "c"]
values = [1, 2, 3]
result = {k: v for k, v in zip(keys, values)}
# {"a": 1, "b": 2, "c": 3}

üéØ Set Comprehension

üìù Basic Syntax

Python 
{expression for item in iterable}

üéØ Examples

Python 
# Unique squares
nums = [1, -1, 2, -2, 3]
unique_squares = {x**2 for x in nums}
# {1, 4, 9}

# Unique first letters
words = ["apple", "ant", "banana", "berry"]
first_letters = {w[0] for w in words}
# {"a", "b"}
