While Loops

Loop while a condition is true! üîÅ

üéØ What are While Loops?

A while loop repeats a block of code as long as a condition is True. Unlike for loops, you don't need to know how many times to repeat beforehand!

üåç Real-Life Analogy

Think of it like eating chips:

  • WHILE there are chips in the bag
  • Take one chip
  • Eat it
  • Check again if there are more chips

You keep eating WHILE chips remain!

üìã For Loop vs While Loop

For Loop While Loop
Know the number of iterations Don't know how many times
Iterate over a sequence Loop based on a condition
for i in range(5): while count < 5:
Automatically moves to next item You control when to stop

üí° When to Use While Loops?

  • When you don't know how many iterations needed
  • When waiting for user input
  • When running until a condition changes
  • For games that run until player loses

üîπ Basic While Loop

üìù Syntax

Python 
while condition:
    # code to execute while condition is True
    # Don't forget to update the condition!

üéØ Example: Counting Up

Python

while count <= 5:
    print(f"Count: {count}")
    count += 1  # IMPORTANT: Update the variable!

print("Done!")

# Output:
# Count: 1
# Count: 2
# Count: 3
# Count: 4
# Count: 5
# Done!

‚ö° Loop Control: break & continue

These keywords give you more control over your loops!

üõë The break Statement

break immediately exits the loop, skipping any remaining iterations.

Python 
# Find the first even number
numbers = [1, 3, 5, 4, 7, 8]
i = 0

while i < len(numbers):
    if numbers[i] % 2 == 0:
        print(f"Found even: {numbers[i]}")
        break  # Exit loop immediately!
    i += 1

# Output: Found even: 4

‚è≠Ô∏è The continue Statement

continue skips the rest of the current iteration and moves to the next one.

Python 
# Print only odd numbers
num = 0

while num < 10:
    num += 1
    
    if num % 2 == 0:
        continue  # Skip even numbers
    
    print(num)

# Output: 1, 3, 5, 7, 9

üéØ Example: Password Checker with break

Python 
correct_password = "python123"
attempts = 3

while attempts > 0:
    password = input("Enter password: ")
    
    if password == correct_password:
        print("‚úÖ Access granted!")
        break
    else:
        attempts -= 1
        print(f"‚ùå Wrong! {attempts} attempts left")

if attempts == 0:
    print("üîí Account locked!")

üîÑ break vs continue Summary

break

  • Exits the loop completely
  • Code after loop runs next
  • Use when you found what you need

continue

  • Skips current iteration only
  • Loop continues normally
  • Use to skip unwanted values

üî∑ The while-else Clause

Python has a unique feature: an else clause with while loops! The else block runs only if the loop completes normally (without hitting break).

üìù Syntax

Python 
while condition:
    # loop code
    if something:
        break  # else won't run
else:
    # runs ONLY if loop wasn't broken

üéØ Example: Search with else

Python 
numbers = [1, 3, 5, 7, 9]
target = 4
i = 0

while i < len(numbers):
    if numbers[i] == target:
        print(f"Found {target}!")
        break
    i += 1
else:
    print(f"{target} not found in list")

# Output: 4 not found in list

üéØ Example: Prime Number Checker

Python 
num = 17
divisor = 2

while divisor < num:
    if num % divisor == 0:
        print(f"{num} is NOT prime (divisible by {divisor})")
        break
    divisor += 1
else:
    print(f"{num} IS prime! ‚ú®")

# Output: 17 IS prime! ‚ú®

üí° When to Use while-else?

  • Searching: else runs if item wasn't found
  • Validation: else runs if all checks passed
  • Any loop where you need to know if it completed naturally

‚ö†Ô∏è Infinite Loops

An infinite loop runs forever because the condition never becomes False. Sometimes this is a bug, but sometimes it's intentional!

üêõ Accidental Infinite Loop (Bug!)

Python - DON'T DO THIS! 
# ‚ùå WRONG - count never changes!
count = 1
while count <= 5:
    print(count)
    # Forgot: count += 1
    # This will print 1 forever! üò±

üÜò How to Stop an Infinite Loop

If your program is stuck in an infinite loop:

  • In terminal: Press Ctrl + C
  • In IDE: Click the "Stop" button

‚úÖ Intentional Infinite Loop (with break)

Sometimes you WANT an infinite loop, with a way to exit:

Python 
# ‚úÖ Menu system with intentional infinite loop
while True:
    print("\nüìã MENU")
    print("1. Say Hello")
    print("2. Say Goodbye")
    print("3. Exit")
    
    choice = input("Enter choice: ")
    
    if choice == "1":
        print("Hello! üëã")
    elif choice == "2":
        print("Goodbye! üëã")
    elif choice == "3":
        print("Exiting...")
        break  # Exit the infinite loop
    else:
        print("Invalid choice!")

üí° while True Pattern

while True: with break is a common pattern for:

  • Menu-driven programs
  • Games (play until quit)
  • Input validation (keep asking until valid)
  • Servers (run forever until shutdown)
