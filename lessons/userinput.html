âŒ¨ï¸ The input() Function

The input() function lets your program ask the user for information!

ğŸŒ Real-Life Analogy

Think of input() like asking someone a question and waiting for their answer. The program pauses until they type something and press Enter!

ğŸ“ Basic Syntax

Python

# Example

ğŸ” How It Works

  1. Python displays the message in quotes (the "prompt")
  2. The program waits for the user to type
  3. User presses Enter
  4. Whatever they typed gets stored in the variable
Python

print(f"Nice to meet you, {name}!")

# The program shows:
# Enter your name: Alex     â† User types "Alex"
# Nice to meet you, Alex!   â† Program responds
