‚å®Ô∏è The input() Function

The input() function lets your program ask the user for information!

üåç Real-Life Analogy

Think of input() like asking someone a question and waiting for their answer. The program pauses until they type something and press Enter!

üìù Basic Syntax

Python 
# Ask for input and store it in a variable
answer = input("Your question here: ")

# Example
name = input("What is your name? ")
print("Hello, " + name + "!")

üîç How It Works

  1. Python displays the message in quotes (the "prompt")
  2. The program waits for the user to type
  3. User presses Enter
  4. Whatever they typed gets stored in the variable
Python 
# Simple greeting program
name = input("Enter your name: ")
print(f"Nice to meet you, {name}!")

# The program shows:
# Enter your name: Alex     ‚Üê User types "Alex"
# Nice to meet you, Alex!   ‚Üê Program responds

‚ö†Ô∏è Important: input() ALWAYS returns a string!

Even if the user types a number, Python treats it as text:

age = input("Enter your age: ")  # User types: 15
print(type(age))  # <class 'str'> - It's a STRING!

üîÑ Converting Input to Numbers

Since input() always returns a string, you need to convert it to do math!

üìù Convert to Integer

Python 
# ‚ùå Wrong - this won't work for math!
age = input("Enter your age: ")
next_year = age + 1  # ERROR! Can't add string + number

# ‚úÖ Correct - convert to int first!
age = int(input("Enter your age: "))
next_year = age + 1  # Works! age is now a number
print(f"Next year you'll be {next_year}")

üìù Convert to Float

Python 
# For decimal numbers, use float()
price = float(input("Enter the price: "))
tax = price * 0.10
total = price + tax
print(f"Total with tax: ${total}")

üîß Common Pattern

Python 
# Wrap input() with the conversion function
name = input("Name: ")           # String (no conversion)
age = int(input("Age: "))        # Integer
height = float(input("Height: ")) # Float
